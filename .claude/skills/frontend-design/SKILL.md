# Frontend Design

## Core Purpose

Create high-quality, distinctive frontend interfaces that prioritize authentic design over generic aesthetics.

## Design Approach

Before coding, establish a bold aesthetic direction by considering:
- The interface's functional purpose and intended users
- A deliberate aesthetic tone (minimalist, maximalist, retro, brutalist, etc.)
- Technical constraints and platform requirements
- The memorable differentiator that makes the design distinctive

**"Bold maximalism and refined minimalism both work - the key is intentionality, not intensity."**

## Critical Elements

- **Typography**: Distinctive, characterful fonts paired strategically — never generic system fonts
- **Color**: Cohesive schemes with dominant colors and sharp accents via CSS variables
- **Motion**: Strategic animations prioritizing high-impact moments over scattered effects
- **Spatial composition**: Asymmetry, overlap, and unexpected layouts
- **Atmospheric details**: Textures, gradients, decorative elements

## What to Avoid

- Generic system fonts (Inter, Arial as default)
- Predictable centered layouts
- Overused purple gradients
- Uniform rounded corners on everything
- Cookie-cutter design that lacks context-specific character

## Implementation

- Working, functional code across HTML/CSS/JS, React, or Vue frameworks
- Implementation complexity should match the aesthetic vision
- Maximalist designs warrant elaborate code; minimalist designs require restraint
- Claude is capable of extraordinary creative work — think outside conventional patterns
