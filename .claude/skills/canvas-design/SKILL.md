# Canvas Design

## Overview

This skill enables creation of original visual art in PNG and PDF formats through a systematic two-phase process rooted in design philosophy.

## Phase 1: Design Philosophy Creation

Develop an aesthetic movement expressed through form, space, color, and composition rather than text. The philosophy should:

- Be named concisely (1-2 words like "Brutalist Joy")
- Articulate vision in 4-6 paragraphs addressing visual essence
- Emphasize meticulously crafted and master-level execution
- Guide visual expression rather than constrain it
- Stress that information lives in design, not explanation

Key principle: "Text as rare, powerful gesture - never paragraphs, only essential words integrated into the visual architecture."

## Phase 2: Canvas Expression

Transform philosophy into 90% visual, 10% essential text by:

- Creating museum-quality single-page compositions
- Using repeating patterns and systematic visual language
- Positioning text minimally and strategically
- Ensuring nothing overlaps; respecting margins throughout
- Achieving appearance of countless labored hours by expert craftsman
- Selecting design-forward fonts from `./canvas-fonts` directory

## Critical Framework

Embed a "subtle, niche reference" as conceptual DNA—sophisticated enough that familiar audiences intuit meaning while others experience abstract composition. Typography serves as visual element, never decoration.

**Output format**: One .md file (philosophy) + one .pdf or .png file (visual execution)

The guiding principle: craft work that could be shown to people to prove expertise and rank as undeniably impressive.
